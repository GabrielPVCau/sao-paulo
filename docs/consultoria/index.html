Perguntas Frequentes

Este protocolo é desenhado para empresas que já validaram seu produto (Product-Market Fit) e faturam acima de R$ 50k/mês. Se você busca tração inicial, este não é o momento. Se busca escala e eficiência, é para você.
O único risco é continuar como está. Trabalhamos com contratos baseados em sucesso (Success Fee). Se não atingirmos os KPIs estipulados em contrato, nós não ganhamos a variável. Estamos no mesmo barco.
O investimento é composto por um Setup Fee (para cobrir custos de tecnologia e equipe) + Variável sobre o crescimento gerado. O valor exato depende do tamanho do desafio, mas o ROI projetado é sempre positivo no curto prazo.
